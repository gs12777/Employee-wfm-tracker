# Workforce Productivity Analytics Dashboard



## Project Overview



This project analyzes workforce productivity and operational efficiency through an interactive Power BI dashboard.



Datasets were collected from Kaggle and an additional public source, then merged, cleaned, and optimized into a final structured dataset suitable for analysis.



The dashboard highlights employee performance, work-hour trends, and productivity metrics through visual reporting.



---



## Tools \& Technologies Used



- Power BI

- Data Cleaning \& Transformation

- Dataset Integration

- Excel / CSV Handling

- Business Intelligence Reporting



---



## Dashboard Features



- Pie Chart for workforce category distribution

- Line Plot for productivity trends

- 3 KPI Cards for performance indicators

- Employee Performance Table

- Horizontal Bar Chart for work-hour comparison

- Productivity Ranking Analysis



---



## Key Insights



- Employee 5 recorded the highest working hours at 214 hours

- Employee 23 achieved the highest productivity score of 96.4%

- Employee 42 showed lower productivity at 61.7%

- Employee 37 logged the lowest work hours at 118 hours



---



## Business Value



This dashboard enables organizations to identify performance trends, optimize workforce allocation, and support data-driven decision-making.
